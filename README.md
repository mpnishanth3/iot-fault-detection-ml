# iot-fault-detection-ml

# 🔥 IoT Fault Detection using Machine Learning

## 📌 Overview

This project predicts equipment faults using sensor data such as **Temperature and Current**.

It demonstrates how Machine Learning models can be applied to **industrial automation and predictive maintenance**.

---

## 🧠 Models Implemented

* Logistic Regression (Baseline model)
* Decision Tree (Explainable AI)
* Random Forest (Production-level model)

---

## ⚙️ Features

* Real-time fault prediction
* Feature importance analysis
* Multiple model comparison
* Clean modular code structure

---

## 📊 Model Performance

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 1.0      |
| Decision Tree       | 1.0      |
| Random Forest       | 1.0      |

> Note: Dataset is small. Real-world performance improves with more data.
